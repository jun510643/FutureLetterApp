# Future-Letter-Web

\# Future Letter Web App



Write letters for the future and read them on a chosen date.



\## Features (v1.0.0)

\- Write a letter

\- Choose a future date

\- Save and display letters when the date arrives



\## How to Use

1\. Open `index.html` in a browser

2\. Write your letter and select a date

3\. Click "Save Letter"

4\. Letters will appear when the date arrives
